// Brand Name
CYGNUS LOOP
宇宙補給船「シグナス」と、循環するサイクルを組み合わせた名前。

NASAの無人補給船シグナスが、ISSへ必要なものを自動で届け続けるように——Cygnus LOOPは、サロンに必要なものを自動で循環させる。

発注・在庫・補充のループが止まらない。美容師はただ、お客様と向き合えばいい。
Cygnus NASA 無人補給宇宙船
国際宇宙ステーションへの自動補給ミッションに使われる宇宙船の名前。
自律的に・確実に・止まることなく届ける。
その姿がこのツールの在り方と重なる。
LOOP 英語 / 循環・繰り返し
在庫が減る → 自動発注 → 補充される、という一連のサイクルを表す。
止まらない循環が、サロンの安定を支える。

// What We Automate
∞
自動発注
POSデータと在庫をリアルタイムに連携。規定数を下回ったら自動でディーラーへ発注。発注漏れも過剰在庫も、もう起きない。
◎
在庫管理
メニューごとの使用量をPOSから自動算出。今日の施術で何がどれだけ使われたか、リアルタイムで把握できる。
↗
売上・タスク管理
各店舗の売上を一画面で確認。スタッフのタスク進捗も可視化。オーナーの「把握コスト」を限りなくゼロに近づける。
